Пример использования библиотеки `litera5-api-js-client`
=======================================================

- `index.js` - пример подключения для JavaScript
- `cfg.example.js` - пример файла с настройками (экземпляр `Litera5ApiConfig`)

Предварительная настройка
-------------------------

Скопируйте файл `cfg.example.js` -> `cfg.js`, укажите в нём полученные при регистрации в Литере идентификатор
компании и секретный ключ для API.

Команды
-------

- `yarn install` - установить компоненты
- `yarn run example` - запустить пример JavaScript
